# 🚀 Cheer ET — Streamer Donation Platform

Cheer ET is a modern, real-time donation platform designed for content creators, streamers, and digital personalities. It provides a seamless way for audiences to support creators financially while delivering a powerful dashboard experience for managing donations, analytics, and engagement tools.

Built with performance, simplicity, and scalability in mind, Cheer ET brings a professional fintech-style experience to the creator economy.

---

##  Overview

Cheer ET enables streamers to receive donations from their audience in a fast, transparent, and interactive way. The platform focuses on real-time data, smooth user experience, and modern UI design inspired by leading streaming and financial applications.

It is optimized for both desktop and mobile environments, ensuring accessibility across devices.

---

##  Core Features

###  Real-Time Donation System
- Instant donation tracking
- Automatic balance calculation with platform fee deduction
- Live updates without refresh

###  Advanced Analytics
- Daily donation tracking (1–30 days)
- Visual chart representation
- Total raised amount and donor count
- Top donors ranking system

###  Live Interaction Tools
- Sound alerts for incoming donations
- Real-time donation feed
- Toggle-based control for donation and TTS features

###  Smart Notifications
- Browser push notifications for feature changes
- Instant feedback when enabling/disabling key features

###  Creator Controls
- Enable/disable donations
- Enable/disable text-to-speech (TTS)
- Dashboard-based management

###  User Profile System
- Personalized dashboard experience
- Username-based identity
- Profile image support

###  Withdrawal System
- Dedicated withdrawal flow
- Balance tracking and availability display
